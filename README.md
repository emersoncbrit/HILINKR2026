# Hilinkr

Plataforma de afiliados: produtos, campanhas, loja e link na bio.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Desenvolvimento local

```sh
git clone <YOUR_GIT_URL>
cd Hilinkr
npm i
npm run dev
```

## Deploy

O projeto está configurado para deploy na Vercel. Domínio: https://hilinkr.com/
